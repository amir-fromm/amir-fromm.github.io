Notes from class #2 in advanced python course

  • JSON file: JSON is a Java script object notation; equivalent to a dictionary in python, but with a java format.
  • git init: initiate a git repository in an existing directory
  • .gitignore: a file with a list of files to be ognored by git, when updating a repisitory
  • git add .gitignore # add the gitignore file
  • git commit -m "notes" # if updated, the gitignore need to be committed.
  • git push # push changes
  • git reset # remove file from status area
  • git pull # pulling whatever changes were made on the cloud
  • git checkout X # go back to specific version
  • git checkout main # go back to the main version (up to date)
  • open # a commands to open an html file in a linux terminal
